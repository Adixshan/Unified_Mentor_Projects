# 🌦️ Weather App

A simple and beautiful weather application built with **HTML**, **CSS**, and **JavaScript**, using the **Visual Crossing Weather API**.

## 🚀 Features
- Search by city name
- Real-time temperature data (in °F)
- Wind speed info
- Clean and responsive UI
